# Resolution Bundler

Resolution Bundler helps you actually finish your New Year’s resolutions by pairing every goal with a reward that unlocks only after completion.

**[➡️ Use the app](https://temptation-bundler.vercel.app/)**

---

## How it works

You write goals like:

`Domain | Resolution | Reward`

Example:

Fitness | 10 martial arts classes | visit a new destination  
Personal | Build 1 side project | go to a film festival  
Leisure | Make 3 road trips | food tour

When you mark a goal complete:

- the reward is revealed
- the goal moves to the Completed view so you can review what you earned

## Features

![Interface](assets/interface.png)

- Privacy-first: everything stays in your browser
- Create goal + reward bundles
- Active and Completed views

- Edit goals
